# React / Node Todo

## Installation

```bash
# clone repo
git clone https://github.com/tomekz/react-node-todo.git

# navigate to repo
cd react-node-todo

# install deps
npm install

# Run in terminal
npm run serve

# Navigate to http://localhost:3000
```

## Production

```bash
# Override defaults using `server/config/prod.js` first.
npm run serve-prod
```
